# Penamaan Variabel Komponen UI

### Button
Menggunakan awalan (prefix) `btn`

Contoh:
```java
JButton btnSimpan;
JButton btnReset;
```

### Text Field
Menggunakan awalan `txt`

Contoh:
```java
JTextField txtNama;
JTextField txtAlamat;
```

### Label
Menggunakan awalan `lbl`

Contoh:
```java
JLabel lblNama;
JLabel lblAlamat;
```
